# 🌍 Planets AR – A Guide to Our Solar System

An **Augmented Reality (AR)** educational mobile application developed using **Unity** and **AR Foundation**. The application enables users to visualize realistic 3D models of all **eight planets** of the Solar System in the real world using AR plane detection.

---

## 📖 Overview

Planets AR provides an interactive way to learn about the Solar System by allowing users to place and observe 3D planets in their surroundings through their smartphone camera. Users can select any planet from the home screen and explore it in an immersive AR environment.

## 📱 Application Workflow

1. Launch the application.
2. Select a planet from the home screen.
3. Grant camera permission.
4. Scan the surrounding environment.
5. Detect a flat surface using AR.
6. Place the selected planet in the real world.
7. Move around the planet to observe it from different angles.

---

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/<your-username>/PlanetsAR.git
```

2. Open the project in **Unity 6**.

3. Install the required packages:
   - AR Foundation
   - ARCore XR Plugin

4. Switch the build platform to **Android**.

5. Build and run the application on an ARCore-supported Android device.
